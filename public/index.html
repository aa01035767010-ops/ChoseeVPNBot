Как подключить VPN?

1

Скачайте приложение-клиент

Установите программу для импорта VLESS-ключей на ваше устройство:

Android (v2rayNG) iOS / macOS (FoXray) iOS Альтернатива (Streisand) Windows (v2rayN)
2

Скопируйте ключ

Перейдите на вкладку «Устройства» и нажмите синюю кнопку копирования ключа (`vless://...`) для нужного девайса.

3

Импортируйте ключ и подключитесь

Откройте скачанное приложение:

  • В v2rayNG: Нажмите на ➕ сверху -> Выберите «Импорт профиля из буфера обмена» -> Нажмите круглую кнопку с логотипом V внизу для запуска.
  • В FoXray / Streisand: Нажмите кнопку импорта/добавления -> Выберите импорт из буфера обмена (Clipboard) -> Активируйте соединение переключателем.

Новое устройство

Придумайте понятное название для ключа доступа, например: Мой iPhone или Рабочий ПК.
